## 概要

このアプリケーションは、日々のタスクを効率的に管理するためのシンプルなタスク管理ツールです。
学習目的で開発されており、基本的な CRUD 操作の実装を学ぶことができます。

## 主な機能

- ✅ タスクの作成、表示、更新、削除
- ✅ タスクの優先度管理
- ✅ シンプルで使いやすいインターフェース
- ✅ ローカルストレージでのデータ保存

## 技術スタック


- **Frontend:** HTML, CSS, TypeScript
- **Backend:** Node.js + Express
- **Database:** MySQL（ローカル）
- **Framework:** React
